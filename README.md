## ThoughtWorks Docker Image: etcd

[![](http://dockeri.co/image/baselibrary/etcd)](https://registry.hub.docker.com/u/baselibrary/etcd/)

### Base Docker Image

* `latest`: etcd 2.2
* `2.2`   : etcd 2.2
* `2.1`   : etcd 2.1
* `2.0`   : etcd 2.0

### Installation

    docker pull baselibrary/etcd

### Usage

    docker run -it --rm baselibrary/etcd
